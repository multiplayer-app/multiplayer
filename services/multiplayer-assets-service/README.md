# Multiplayer Assets Service

The multiplayer-assets-service is responsible for generating and managing visual assets related to multiplayer sessions.

## Development

```bash
npm run dev
```

## Building

```bash
npm run build
```

## Testing

```bash
npm test
```

ENV variables:
PORT
CORS_DOMAIN
API_PREFIX
SWAGGER_ENABLED
AWS_REGION
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
